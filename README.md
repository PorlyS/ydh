# 永定河与大兴地貌演化互动动画

双击 `打开永定河沙盘.bat`，或直接打开 `index.html` 即可使用。当前版本已经改成纯动画互动叙事，不再使用在线地图底图。

## 现在是什么

这是一个面向课堂的“地理动画游戏”：

- 全屏展示大兴区示意轮廓与永定河路径。
- 永定河水流持续动画，泥沙、平原、沙地、湿地会随章节逐渐显现。
- 点击发光热点触发事件：永定河出山、平原铺展、沙地留下、泉眼冒出、团河行宫、大兴机场、生态修复。
- 下方时间线可跳转章节，也可以自动播放。
- 课堂任务会记录学生是否点亮完整故事链。
- 支持关闭动画和标注，方便讲解或投屏。

## 推荐课堂讲法

1. 从“永定河出山”开始，让学生观察水流如何携带泥沙进入平原。
2. 切到“平原铺展”，解释冲积平原和农业/聚落基础。
3. 切到“沙地留下”，讨论河漫滩、风沙和沙质地的关系。
4. 点击“泉眼冒出”，引出低洼汇水、湿地和稳定水源。
5. 点击“团河行宫”，讨论自然水文条件如何影响皇家园林选址。
6. 点击“大兴机场”，讨论开阔平原如何承载大型交通设施。
7. 最后切到“生态修复”，让学生总结自然过程与人类活动的因果链。

## 操作

- 点击发光热点：触发地点事件
- 点击底部时间线：切换章节
- `播放`：自动播放故事
- `上一章 / 下一章`：手动切换
- `聚焦事件`：点亮当前事件
- `动画开 / 动画关`：控制水流粒子与局部动效
- `标注开 / 标注关`：控制地图文字
- `← / →`：上一章 / 下一章
- `Space`：播放 / 暂停

## 说明

当前大兴区轮廓、河道和地貌范围为教学示意，不是精测 GIS 数据。团河行宫、机场等节点用于课堂叙事串联，正式史料课件建议再配权威文献或图片材料。
